## Key decision: no fabricated deep-link URL builder

The plan text floated a `westlawLinkForCitation()` URL builder. Research (Open Question 5, REFUTED
across 13 years of sources) shows **no composable vendor deep-link URL exists** â€” Westlaw/Lexis
links are opaque UI-generated permalinks. Fabricating a URL formula would itself manufacture a
wrong-link/false-"verified" outcome, so it is intentionally NOT built. The honest, buildable
deliverable is the type-enforced verify-vs-link *boundary*, exactly as the research doc's own
"Redesign Proposal" recommends (a `LinkOnlyProvider`-style capability tag).
